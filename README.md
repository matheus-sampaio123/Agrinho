# Projeto Agrinho: Sustentabilidade e Tecnologia

## Objetivo
Este projeto visa conscientizar sobre o uso da água no campo utilizando ferramentas interativas.

## Tecnologias Utilizadas
- HTML5 (Semântico)
- CSS3 (Flexbox e Media Queries)
- JavaScript (Manipulação de DOM)

## Como usar
Basta abrir o site, digitar a quantidade de litros e ver o cálculo automático. Utilize o botão no topo para alternar entre modo claro e escuro.
